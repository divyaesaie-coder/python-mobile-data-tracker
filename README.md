# python-mobile-data-tracker
A simple Python program to track remaining mobile data usage daily and display the available balance after each usage.
# Python Mobile Data Tracker

A beginner-friendly Python project that tracks daily mobile data usage and shows remaining data balance.

## Features
- Track daily mobile data usage
- Display remaining data balance
- Prevent invalid data usage input
- Continue or stop tracking anytime

## Technologies Used
- Python 3

## How It Works
The program starts with a fixed mobile data amount and:
1. Takes used data input from the user
2. Deducts used data from total balance
3. Displays remaining data
4. Stops when data finishes or user exits

## Concepts Used
- while loop
- if-else conditions
- Arithmetic operations
- User input handling

```bash
